This is a Node js, EJS and SQLite based project in which 

1. EJS view engine is used to show Product Master and Category Master Page
2. Node js used to make backend api for CRUD operations
3. SQLite used as a RDBMS based database for storing the data in localhost
4. Server side pagination is used in the product page such that each page will only show 10 product
5. Each product belongs to a existing category with unique productId as well as categoryId

Use "node db-init.js" to initialize the database
and then use "node app.js" to start the project 
"http://localhost:3000" will be the localhost website to see how the website works
